Instructions

  • The fCC Issue Tracker allows users to create and update issues assigned to specific projects.
  • To get a list of all issues related to a project, users can use the endpoint: /api/issues/{project name}
  • This list can be filtered by passing parameters in the query, for example: /api/issues/{project name}?close=true
  • Users can also view and add issues to a specific project by visiting:
    /{project name}
